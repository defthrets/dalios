# 🧠 Dalios

AI Trading Analyst built on Ray Dalio's systematic investment principles.

## Features
- 220+ assets (crypto, equities, commodities, ASX, EU, Asia)
- 20 pattern recognition systems
- 11 AI provider integrations
- Interactive geopolitical conflict map (136 countries)
- Markowitz portfolio optimizer
- Full system backtest engine
- Multi-country debt cycle analysis (US, CN, EU, JP, AU, GB)
- AI chat assistant with portfolio context
- 16 broker integrations

## Quick Start
```bash
npm install
npm run dev
```

## Deploy
```bash
npm run build
# Deploy the dist/ folder to any static host
```
